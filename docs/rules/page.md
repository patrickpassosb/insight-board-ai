## Details

**[EVENTO PRESENCIAL EM SÃO PAULO]**

_Atenção: as inscrições para este evento são realizadas pela página: [https://www.liferay.com/pt/web/lr/br-26-liferay-marketplace-hackathon](https://www.liferay.com/pt/web/lr/br-26-liferay-marketplace-hackathon). Inscrições realizadas pela plataforma Meetup não serão computadas para este evento._

Junte-se a nós em São Paulo para um hackthon presencial para construir fragmentos para usar no Liferay.

O Liferay Marketplace é o seu destino oficial para apps, temas, fragmentos e soluções projetadas para expandir os limites do Liferay DXP. Agora, chegou a sua vez de criar a próxima grande solução para o nosso ecossistema!

**Participe do primeiro Hackathon de Fragmentos do Liferay Marketplace!**

Vamos reunir desenvolvedores de todo o Brasil para passar um dia desenvolvendo fragmentos a serem usados no Liferay, com demonstrações ao vivo, prêmios e uma ponte direta para a comunidade Liferay no Brasil.

 Vagas limitadas! [Registre-se aqui para garantir sua vaga!](https://www.liferay.com/pt/web/lr/br-26-liferay-marketplace-hackathon)

**O desafio**

 Mestres dos Fragmentos: arquitetando o futuro dos portais - sua missão é construir um Fragmento de alto desempenho e reutilizável com componentes visualmente impressionantes, e que também resolvam desafios reais.

**Quem pode participar**

 Este evento é para você que já usa Liferay e já quis agilizar seu desenvolvimento com fragmentos pré-prontos. Não é necessário ter conhecimento prévio sobre o Liferay Marketplace, esse é o lugar para aprender como ele funciona e lançar seu primeiro fragmento.
