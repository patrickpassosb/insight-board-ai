## Problema
Dashboards tradicionais exibem números, mas não ajudam o usuário a interpretar rapidamente o cenário nem a decidir a próxima ação. Em contextos operacionais e executivos, isso aumenta o tempo entre observar o dado e agir.

## Funcionalidade
Criamos uma coleção modular de fragmentos para Liferay que permite montar dashboards reutilizáveis com KPIs, gráficos e um painel de insights com IA. A solução combina:

- visualização de dados com Chart.js;
- fragmentos configuráveis via Page Editor;
- componentes reutilizáveis para compor diferentes páginas;
- análise de métricas com IA para gerar resumo executivo, nível de risco, insights e recomendação acionável.

## Sugestões de Ícones (Material Symbols)

  Como seu dashboard é focado em dados e IA, aqui estão opções que transmitem profissionalismo e clareza:

   * Para o KPI Card (Financeiro/Vendas): attach_money, payments, trending_up, show_chart, analytics, receipt_long.
   * Para o KPI Card (Usuários/Pessoas): person, group, groups, person_search, account_circle.
   * Para o KPI Card (Churn/Risco): trending_down, warning, report_problem, power_off (se for inatividade).
   * Para o módulo de IA: psychology, smart_toy, auto_awesome, insights, rocket_launch.
   * Para Ações (Geral): download, refresh, settings, edit_square, chevron_right.

  Dica de design: Use ícones do estilo "Outlined" para todos eles. Isso mantém a consistência visual, um detalhe que os avaliadores técnicos (especialmente da Liferay) notam imediatamente.
