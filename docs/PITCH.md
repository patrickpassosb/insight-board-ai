# 3-Minute Pitch Outline

## Structure (3 minutes total)

### 1. The Problem (30 sec)
- Companies use Liferay portals to manage operations
- Dashboards show data, but executives still need to *interpret* it themselves
- "What does this 5% churn rate actually mean? What should I do about it?"

### 2. The Solution (30 sec)
- **InsightBoard AI** — a drop-in Liferay Fragment that turns raw metrics into strategic decisions
- It doesn't just show numbers — it *analyzes* them using AI and tells you what to do next
- Zero configuration: install from Marketplace, add to any page, instant value

### 3. Live Demo (60 sec)
- Show the dashboard with KPI cards and charts
- Click "Generate AI Insights"
- Show the AI analyzing data in real time
- Highlight: executive summary, risk level, specific recommendations
- Show an actionable button (e.g., "Draft Email to Sales Team")

### 4. How It Works (30 sec)
- Quick architecture diagram: Fragment → Node.js API → Mistral AI → Structured response
- API key secured server-side
- Fragment is pure HTML/CSS/JS — fully compatible with Liferay Marketplace

### 5. Business Value (30 sec)
- Any Liferay customer can install this today
- Reduces decision-making time from hours to seconds
- Extensible: swap mock data for real Liferay Objects data
- Scales to any industry: sales, HR, operations, support

## Key Phrases to Use
- "From data to decisions in one click"
- "Your dashboard doesn't just report — it thinks"
- "Drop-in intelligence for any Liferay portal"

## Demo Checklist
- [ ] Liferay running with Fragment deployed
- [ ] Node.js API running on port 3000
- [ ] Mistral API key configured
- [ ] Test the full flow at least twice before presenting
- [ ] Have a backup screenshot in case of network issues
