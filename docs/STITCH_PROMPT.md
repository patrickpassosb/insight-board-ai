# Stitch Prompt

Copy and paste the prompt below into Google Stitch:

---

Design a business analytics dashboard called "InsightBoard AI". This is a single-page dashboard fragment that will be embedded inside a Liferay portal. It should look modern, professional, and clean.

## Layout (top to bottom):

### Header
- Title "InsightBoard AI" on the left
- Subtitle: "Turning Data into Decisions with AI"
- A small badge or label showing the current period: "March 2026"

### Top Row — 4 KPI Cards in a horizontal row
Each card has: an icon, a label, a large number, and a small percentage change indicator (green for positive, red for negative).
1. "Revenue" — $125,000 — down 5.2%
2. "Growth" — 12.3% — up 2.1%
3. "Active Users" — 1,200 — up 8.4%
4. "Churn Rate" — 4.8% — up 1.2% (this is bad, so show it in red)

### Middle Row — 2 columns side by side
**Left column (wider, ~60%):** A line chart showing "Revenue Trend" over 6 months (Oct, Nov, Dec, Jan, Feb, Mar) with values going up and then dipping in the last month. Below it, a bar chart showing "Support Tickets by Month" for the same 6 months.

**Right column (~40%):** A doughnut chart showing "Revenue by Segment" with 4 segments (Enterprise 45%, Mid-Market 30%, SMB 15%, Startup 10%).

### Bottom Section — AI Insights Panel
This is the most important visual section. It should stand out from the rest of the dashboard with a slightly different background (subtle gradient or accent border).

It contains:
- A header: "AI-Powered Insights" with a sparkle/AI icon
- A prominent button: "Generate Insights" with a magic wand or sparkle icon
- Below the button, a results area showing:
  - **Executive Summary**: "Revenue is down 5.2% due to increased churn. The support ticket spike in February correlates with the v3.2 release."
  - **Risk Level**: A badge showing "High" in red/orange
  - **Key Insights** (2 bullet points):
    - "Churn rate (4.8%) is 2x above the industry average of 2.4%"
    - "Support ticket volume increased 40% after the February product release"
  - **Recommendation**: "Initiate a customer retention campaign immediately and audit the v3.2 release for critical bugs."
  - An action button: "Draft Email to Sales Team" (this represents an actionable suggestion from the AI)

## Design Guidelines:
- Use a card-based layout with subtle shadows and rounded corners
- Use a clean sans-serif font
- Charts should have smooth lines and modern styling
- The AI Insights panel should feel special — it is the hero section of this dashboard
- Responsive grid layout
- Include a loading spinner placeholder near the "Generate Insights" button for when the AI is processing
- All text in English
