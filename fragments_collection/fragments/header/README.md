# InsightBoard Header Fragment

A Liferay fragment representing the primary header of the InsightBoard AI dashboard. Features prominent branding and context-aware styling.

## Setup Requirements
- Requires Liferay 7.4+.
- Can be imported individually or as part of the total `fragments_collection`.

## Editable Elements (Liferay Page Editor)
- `title`: The main headline text (Text).
- `subtitle`: The secondary descriptive subtitle (Text).
- `period`: The right-aligned period badge stating the current working window (Text).
