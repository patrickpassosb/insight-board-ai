# InsightBoard Sidebar Fragment

A Liferay fragment representing the structural left-hand navigation sidebar for the InsightBoard AI Application.

## Setup Requirements
- Optimized for grid-layouts. Intended to be placed in the primary left column of a page structure.
- Requires Liferay 7.4+.

## Editable Elements (Liferay Page Editor)
- `brand-title`: Primary application name.
- `brand-subtitle`: Smaller gray classification text.
- `nav*`: Five editable navigation labels mapping to different views.
- `btn-report`: The text embedded in the bottom call-to-action button.
